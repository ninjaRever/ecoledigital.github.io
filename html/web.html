Introduction au web

Ce cours est conçu pour les débutants souhaitant apprendre les bases du développement web. Vous découvrirez les langages HTML, CSS et JavaScript. À la fin de ce cours, vous serez capable de créer une page web simple et structurée.

Modules du cours

  • Chapitre 1
    1 - Réseau Informatique et Internet
  • Chapitre 2
    2 - Le Web et le concept
  • Chapitre 3
    3 - Site Web et hébergement
  • Chapitre 4
    4 - Le Webmastering et les autres
  • Chapitre 5
    5 - Les langages du Web
  • Chapitre 6
    6 - Les noms de domaine
